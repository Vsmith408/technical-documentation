Introduction

Flexbox is the new way to style your all elements. It can help with everything from aligning elements, to distributing space between items in a container. This is especially helpful when you don't know the size of the element. Flexbox is normally used on small-scale layouts, compared to the grid system which is used on large-scale layouts.

How To Get Started

You begin by going into your CSS file and typing in the parent element you want to call. Once you do this, you will set the element to " display: flex; ". It's that simple! You can then define what properties you want to add to the element. You have now made that item a flex-container. The parent element is known as the flex-container, meaning that is the element you have set to flex. The children in this element are now known as flex-items.

Parent Properties

Below are the different types of properties you can use on a parent element, along with a brief description.

  1. Display : This property allows you to set a flex container
  2. Flex-direction: There are four directions you can use with this property.
    • row: this aligns your items from left to right
    • row-reverse : this aligns your items from right to left
    • column: aligns your items from top to bottom
    • column-reverse: aligns your items from bottom to top

    When you're typing this in your CSS document, it will look like this: 

                
.container {
    flex-direction: row || row-reverse || column || column-reverse;
}
  3. Flex-wrap: This allows you to determine where you want your text to go to the next line. This has three ways to do this:
    • nowrap: all items will be on one line
    • wrap: items will wrap from top to bottom
    • wrap-reverse: items will wrap onto a new lne from bottom to top

    When you're typing this in your CSS document, it will look like this: 

                  
.container {
    flex-wrap: nowrap || wrap || wrap-reverse;
}
Child Properties

Below are the different types of properties you can use on a child element, along with a brief description.

  1. Flex-grow: This property takes a unitless value and determines what portion of the space in the container it should take up. You can enter whatever number you'd like; however, negative numbers are not accepted.

    When you're typing this in your CSS document, it will look like this: 

                  
.item {
  flex-grow: (number);
}
  2. Flex-shrink: This allows the item to shrink as necessary.

    When you're typing this in your CSS document, it will look like this: 

                  
.item {
  flex-shrink: (number);
}
  3. Flex-basis: This determines the default size of the item. You can use a number or a keyword.

    When using this in your CSS document, it will look like this: 

                  
.item {
  flex-basis: length || keyword(auto);
}
  4. Flex: This allows you to shorthand your Flexbox properties. It is recommended to use the shorthand version. However, the flex-shrink and flex-basis properties are not required.

    When using this in your CSS document, it will look like this: 

    .item {
  flex: [ <'flex-grow'> <'flex-shrink'> || <'flex-basis'> ]
}
